.. _snippet-infuse:

Infuse-IoT SDK snippet (infuse)
###############################

.. code-block:: console

   west build -S infuse [...]

Overview
********

Updates to upstream compatible boards to compile for Infuse-IoT and operate in
a low power manner.

Supported Upstream Boards
*************************

.. rst-class:: rst-columns

 - `Nordic Thingy53`_
 - `Nordic nRF5340 DK`_
 - `Nordic nRF52840 DK`_
 - `Nordic nRF9151 DK`_
 - `Nordic nRF9160 DK`_
 - `Nordic nRF9161 DK`_

Supported Infuse Boards
***********************

.. rst-class:: rst-columns

 - `Nordic nRF7002 DK`_
