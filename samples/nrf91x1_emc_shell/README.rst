nRF91x1 EMC Shell
#################

Sample application that exposes the nRF91x1 ``%XRFTEST`` AT command through the Zephyr shell.

.. warning::

    This command transmits power to the selected RF band and can violate the radio
    directives of the region or country. Make sure that the equipment is in an RF-shielded room or
    connected to an RF cable so that RF power does not leak.
